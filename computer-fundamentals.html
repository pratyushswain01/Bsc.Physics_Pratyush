3. What is a binary code and why it is used in a computer?

Binary code is a system that represents text, computer processor instructions, or any other data using a two-symbol system, typically "0" and "1". It is used in computers because their fundamental components (transistors) operate in two states: on or off, which can be directly represented by 1 and 0, respectively. This makes it the most basic language of a computer.

4. What is a computer Algorithm and why is it important in computer science?

A computer algorithm is a finite, step-by-step set of instructions for solving a problem or accomplishing a task. It is important because it provides a clear, unambiguous blueprint for writing a computer program. A well-designed algorithm ensures that the program is correct, efficient, and reliable.

5. What is a computer's cache memory and how does it improve performance?

Cache memory is a small, extremely fast type of volatile memory that provides high-speed data access to the CPU. It improves performance by storing frequently used data and instructions, reducing the time the CPU spends waiting for data from the slower main memory (RAM).

6. What is computer booting?

Booting is the process of starting a computer. It begins when the power is turned on and involves loading the operating system (OS) from the storage drive into the main memory (RAM). The initial instructions for booting are stored in the BIOS or UEFI firmware.

7. Define Register.

A register is a small amount of very fast memory located directly inside the Central Processing Unit (CPU). Registers are used to hold the data, instructions, and addresses that the CPU is currently working on, allowing for extremely quick processing.

8. What role does the ALU play in a computer system?

The ALU (Arithmetic Logic Unit) is a crucial component of the CPU. Its role is to perform all arithmetic operations (like addition, subtraction, multiplication) and logical operations (like AND, OR, NOT). It is the mathematical brain of the computer.

11. What is router and how is it different from repeater?

A router is a networking device that connects different networks and forwards data packets between them based on their IP address. A repeater is a simpler device that only regenerates and retransmits a network signal to extend its range, without any routing intelligence.

12. Write down the importance of networking.

The importance of networking includes:

  • Resource Sharing: Sharing hardware like printers and files.
  • Communication: Enabling email, video conferencing, and messaging.
  • Access to Information: Providing access to remote databases and the internet.
  • Centralized Management: Simplifying software updates and data backups.

13. Define hybrid computer with suitable example.

A hybrid computer is a computer that has features of both analog and digital computers. It combines the speed of analog computers and the memory and accuracy of digital computers. For example, a computer used in a hospital's ICU to monitor a patient's heartbeat (analog) and then convert the data into numbers and values (digital) is a hybrid computer.

16. What is Gateway in computer network?

A gateway is a networking device that acts as an entry/exit point for a network, connecting two dissimilar networks that use different communication protocols. It translates data from one protocol to another. Your home router acts as a gateway between your local LAN and the internet (WAN).

17. What are storage devices? Write down its function.

Storage devices are computer hardware used to store data and programs permanently (non-volatile). Their function is to retain information even when the computer is powered off, allowing it to be retrieved later. Examples include Hard Disk Drives (HDD), Solid-State Drives (SSD), and USB flash drives.

19. What is volatile memory?

Volatile memory is a type of computer memory that requires power to maintain the stored information. All data is lost when the power is turned off. The most common example is RAM (Random Access Memory).

Section B (5 Marks Questions)

1. What is software? Explain different types of software.

Software is a collection of instructions, data, or programs that tell a computer how to perform specific tasks. It is the logical, intangible part of a computer system. Software is broadly classified into two main types:

1. System Software

System software manages the computer's hardware and provides a platform for application software. It is essential for the computer's operation.

  • Operating System (OS): The core software that manages all hardware and software resources. Examples: Windows, macOS, Linux, Android.
  • Device Drivers: Programs that allow the OS to communicate with specific hardware devices like printers or graphics cards.
  • Utility Software: Programs that help maintain and configure the computer. Examples: Antivirus software, disk cleanup tools.
2. Application Software

Application software (apps) is designed to perform specific tasks for the end-user.

  • Word Processors: For creating documents (e.g., MS Word).
  • Web Browsers: For accessing the internet (e.g., Chrome).
  • Spreadsheet Software: For organizing and analyzing data (e.g., MS Excel).
  • Database Software: For managing databases (e.g., MySQL).

2. What do you mean by internet? Explain with its application.

The Internet is a massive, global network of interconnected computer networks. It uses the standard TCP/IP protocol suite to link billions of devices worldwide, allowing them to communicate and share information. It is a "network of networks" that is not owned by any single entity.

Applications of the Internet:
  1. World Wide Web (WWW): A system of interlinked hypertext documents accessed via the internet. It's how we access websites.
  2. Communication: Email, instant messaging (WhatsApp), video conferencing (Zoom, Google Meet), and social media (Facebook, Instagram).
  3. E-Commerce: Buying and selling goods and services online through platforms like Amazon and Flipkart.
  4. Information Retrieval: Accessing a vast amount of information through search engines like Google.
  5. Entertainment: Streaming music (Spotify), videos (YouTube, Netflix), and online gaming.

3. Explain different types of networks.

Computer networks are classified based on their geographical scope:

  • PAN (Personal Area Network): Spans a very small area, typically around a single person (a few meters). It's used to connect personal devices like a smartphone, laptop, and wireless headphones via Bluetooth.
  • LAN (Local Area Network): Connects devices within a limited area like a home, office building, or school campus. It has high speed and is usually privately owned. Technologies include Ethernet and Wi-Fi.
  • MAN (Metropolitan Area Network): Covers a larger area than a LAN, such as a city. It often interconnects multiple LANs. It can be owned by a single organization or a public provider like a cable company.
  • WAN (Wide Area Network): Spans a large geographical area, like a country or the entire globe. It consists of multiple interconnected LANs and MANs. The Internet is the world's largest WAN. It uses public communication lines and has slower speeds than LANs.

5. What is Register? Explain different types of register with suitable net sketch.

A Register is a small, extremely fast memory location inside the CPU. It holds data temporarily that is being actively processed. Since registers are part of the CPU chip itself, they provide the fastest way for the system to access data.

Types of Registers:
  • Memory Address Register (MAR): Holds the memory address of the data or instruction that needs to be fetched from or stored into memory.
  • Memory Data Register (MDR): Holds the data that is being transferred to or from memory. It acts as a buffer.
  • Program Counter (PC): Holds the memory address of the next instruction to be executed.
  • Accumulator (AC): Stores the result of calculations performed by the ALU.
  • Instruction Register (IR): Holds the current instruction being executed.

7. Briefly explain different input devices with functionalities.

Input devices are hardware peripherals used to provide data and control signals to a computer. Some common input devices are:

  1. Keyboard: The most common input device, used to enter text, numbers, and commands into the computer.
  2. Mouse: A pointing device that detects two-dimensional motion, allowing users to interact with graphical user interfaces (GUIs) by clicking and selecting items.
  3. Scanner: A device that optically scans images, printed text, or objects and converts them into a digital format.
  4. Microphone: An audio input device that converts sound waves into electrical signals, allowing the computer to record voice or other sounds.
  5. Webcam (Web Camera): A video camera that feeds its image in real time to or through a computer, used for video conferencing and streaming.

8. What is firmware? Write down its application.

Firmware is a special type of software that is permanently stored on a hardware device's read-only memory (ROM). It provides low-level control for the device's specific hardware. Unlike regular software, it is not typically changed or updated by the end-user.

Applications of Firmware:
  • Computer BIOS/UEFI: The firmware on a computer's motherboard that initializes the hardware during the booting process.
  • Embedded Systems: In devices like washing machines, microwaves, and traffic lights, firmware acts as the device's operating system.
  • Consumer Electronics: In TVs, remote controls, and digital cameras, firmware controls their basic functions.
  • Network Devices: Routers and switches run on firmware that manages their networking tasks.

9. Briefly explain different output devices with functionalities.

Output devices are hardware peripherals that receive data from a computer and translate it into a human-perceptible form, such as text, graphics, audio, or video.

  1. Monitor (or Display Screen): The most common output device, displaying information in visual form.
  2. Printer: A device that produces a hard copy (a permanent physical representation) of text or graphics on paper.
  3. Speakers: Audio output devices that convert electronic signals from the computer into sound waves.
  4. Projector: A device that takes a video signal from a computer and projects the corresponding image onto a large screen.
  5. Headphones: A personal audio output device worn on or over the ears.

10. What is memory? Explain briefly different types of computer memory.

Computer Memory refers to the physical devices used to store data and programs, either temporarily or permanently. It is one of the fundamental components of a computer system.

Types of Computer Memory:
  1. Primary Memory (Main Memory): This is volatile memory that the CPU can access directly. It holds data and instructions that are currently in use.
    • RAM (Random Access Memory): Volatile memory for active programs and data. Data is lost when power is off.
    • ROM (Read-Only Memory): Non-volatile memory that stores firmware like the BIOS. Data is permanent.
  2. Secondary Memory (Storage): This is non-volatile memory used for long-term storage of data and programs. It is slower than primary memory.
    • HDD (Hard Disk Drive): A magnetic storage device with large capacity.
    • SSD (Solid-State Drive): A faster, more durable storage device that uses flash memory.
    • Optical Disks: CD, DVD, Blu-ray discs.

11. What is operating system? Explain its functionalities.

An Operating System (OS) is the core system software that acts as an intermediary between the computer hardware and the end-user. It manages all the hardware and software resources of the system.

Major Functionalities of an OS:
  • Process Management: Manages the execution of programs (processes), including scheduling and termination.
  • Memory Management: Allocates and de-allocates memory space to programs as needed.
  • File Management: Organizes and manages files and directories on storage devices, controlling their creation, deletion, and access.
  • Device Management: Manages communication with hardware devices through their drivers.
  • Security: Provides security through user authentication (passwords) and protecting system resources from unauthorized access.
  • User Interface: Provides an interface (Command-Line or Graphical) for the user to interact with the computer.

12. Explain about various generation of computer.

The evolution of computers is categorized into five generations:

  1. First Generation (1940s-1950s): Used vacuum tubes as the main component. They were enormous, expensive, consumed a lot of power, and were unreliable. Example: ENIAC.
  2. Second Generation (1950s-1960s): Replaced vacuum tubes with transistors, which were smaller, faster, cheaper, and more reliable. Assembly language was used. Example: IBM 1401.
  3. Third Generation (1960s-1970s): Used Integrated Circuits (ICs), where multiple transistors were placed on a single silicon chip. This made computers smaller and more powerful. High-level programming languages were introduced. Example: IBM System/360.
  4. Fourth Generation (1970s-Present): Used microprocessors, which contain thousands or millions of transistors on a single chip (Very Large Scale Integration - VLSI). This led to the development of personal computers (PCs). Example: Apple Macintosh, IBM PC.
  5. Fifth Generation (Present and Beyond): Focuses on Artificial Intelligence (AI), parallel processing, and quantum computing. These computers are designed to understand natural language and make decisions like humans.

Section C (8 Marks Questions)

1. What is a Network? Explain different types of computer Network.

A computer network is a collection of two or more interconnected computers and other hardware devices (like printers, servers) linked together through communication channels. The primary purpose of a network is to facilitate the sharing of resources (such as files, software, and hardware) and to enable communication (such as email and video conferencing) among users.

Types of Computer Networks

Computer networks are primarily classified based on the geographical area they span. The main types are:

1. LAN (Local Area Network)
  • Area: A LAN connects devices within a limited geographical area, such as a single building, an office, a school, or a home.
  • Ownership: They are typically privately owned and managed by a single organization or individual.
  • Speed: LANs offer very high data transfer speeds (e.g., 100 Mbps to 10 Gbps) because of the short distances involved.
  • Technology: Common technologies are Ethernet (cables) and Wi-Fi (wireless).
2. MAN (Metropolitan Area Network)
  • Area: A MAN covers a larger area than a LAN, such as an entire city. It can be formed by interconnecting multiple LANs.
  • Ownership: A MAN can be owned by a single organization or by a public service provider.
  • Technology: MANs often use fiber optic cables.
3. WAN (Wide Area Network)
  • Area: A WAN spans a very large geographical area, such as a state, a country, or the entire globe. It is a network of networks.
  • Ownership: WANs are rarely owned by a single organization and use public telecommunication channels.
  • Speed: Data transfer speeds are significantly slower than in a LAN.
  • Example: The Internet is the largest WAN in the world.
4. PAN (Personal Area Network)
  • Area: This is the smallest type of network, centered around an individual's personal workspace (a few meters).
  • Technology: Bluetooth and Wi-Fi Direct are common technologies.

2. What do you mean by memory hierarchy? Explain Briefly.

The Memory Hierarchy in a computer system is a structural arrangement of its various memory and storage devices. It is organized in a pyramid-like structure based on three key characteristics: access speed, cost per bit, and storage capacity.

The core principle is that the fastest memories are the most expensive and have the smallest capacity, so they are placed at the top of the hierarchy, closest to the CPU. As you move down the hierarchy, the speed and cost decrease, while the capacity increases.

Levels of the Memory Hierarchy:
  1. Level 0: CPU Registers:
    • Location: Inside the CPU.
    • Characteristics: Fastest access speed, highest cost, smallest capacity. Holds data currently being processed.
  2. Level 1: Cache Memory:
    • Location: On or very close to the CPU chip.
    • Characteristics: Slower than registers but much faster than RAM. Stores frequently accessed data from RAM to reduce access time.
  3. Level 2: Primary Memory (RAM):
    • Location: On the motherboard.
    • Characteristics: Slower than cache, but much faster than secondary storage. It is volatile and holds the OS, active programs, and data.
  4. Level 3: Secondary Storage:
    • Location: Internal or external drives.
    • Characteristics: Slowest access speed, lowest cost per bit, largest capacity. It is non-volatile. Examples include SSDs, HDDs.
  5. Level 4: Tertiary Storage (Offline Storage):
    • Location: External, often removable media.
    • Characteristics: Very slow, used for backups and archiving. Examples include magnetic tapes and optical disk libraries.

3. Explain the classification of computer based on data processed.

Computers can be classified into three main types based on the kind of data they are designed to process:

1. Analog Computers
  • Data Processed: They process analog data, which is continuous in nature and not discrete. This data represents physical quantities like temperature, pressure, voltage, and speed.
  • Functionality: They measure continuous changes in physical quantities. They don't require storage as they measure and process data in real-time.
  • Accuracy: They are less accurate compared to digital computers.
  • Examples: Speedometer of a car, traditional thermometer, seismograph.
2. Digital Computers
  • Data Processed: They process digital data, which is discrete and represented in binary form (0s and 1s).
  • Functionality: They perform calculations and logical operations with high speed and accuracy. They have large memory and storage capacities.
  • Accuracy: They are highly accurate and provide precise results.
  • Examples: Laptops, desktops, smartphones, calculators. All modern computers are digital computers.
3. Hybrid Computers
  • Data Processed: They are designed to process both analog and digital data. They combine the features of both types.
  • Functionality: They accept continuous analog signals, convert them into digital form for processing, and can provide output in either analog or digital form. They combine the high speed of analog computers with the accuracy and memory of digital computers.
  • Accuracy: They offer good accuracy and high processing speed.
  • Examples: Computers used in hospital ICUs to monitor a patient's vital signs (analog) and convert them into digital data for analysis. Computers used in scientific experiments and industrial control processes.

4. Explain different types of networking devices. Differentiate between LAN and WAN.

Types of Networking Devices:
  1. Hub: A basic Layer 1 device that connects multiple computers in a LAN. It broadcasts any incoming data packet to all connected ports, leading to inefficiency and traffic collision.
  2. Switch: A more intelligent Layer 2 device that connects devices in a LAN. It learns the MAC address of each connected device and forwards data packets only to the specific destination port, improving network performance significantly.
  3. Router: A Layer 3 device that connects two or more different networks (e.g., a home LAN to the Internet WAN). It uses IP addresses to determine the best path to forward data packets between networks. Routers are the backbone of the internet.
  4. Repeater: A Layer 1 device used to regenerate and retransmit a weak network signal. It helps to extend the physical range of a network by combating signal degradation (attenuation) over long distances.
  5. Gateway: A versatile device that acts as a bridge between two dissimilar networks using different protocols. It performs protocol conversion to allow communication. A home router, for instance, acts as a gateway between the local network and the ISP's network.
