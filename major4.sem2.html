Easy & Theoretical Focus Set

Subject: Physics (Mathematical Physics) | Total Marks: 100
Part A: Definitions & Statements
(Answer any NINE questions | 9 × 2 = 18 Marks)
  1. Define the Gamma Function, \(\Gamma(n)\), using its integral form.
  2. State Rodrigues' formula for the Legendre Polynomials, \(P_n(x)\).
  3. What are Dirichlet's conditions for the convergence of a Fourier series?
  4. State the generating function for the Legendre Polynomials, \(P_n(x)\).
  5. Write the general form of the one-dimensional wave equation and identify the terms.
  6. State the property of orthogonality for Legendre polynomials over the interval \([-1, 1]\).
  7. What is the value of \(\Gamma(\frac{1}{2})\)?
  8. Define a regular singular point of a second-order linear differential equation.
  9. State the parity relation for Legendre polynomials, relating \(P_n(-x)\) to \(P_n(x)\).
Part B: Standard Derivations & Short Proofs
(Answer any EIGHT questions | 8 × 5 = 40 Marks)
  1. Prove the fundamental recurrence relation for the Gamma function: \(\Gamma(n+1) = n\Gamma(n)\).
  2. Using Rodrigues' formula, derive the expression for the Legendre polynomial \(P_2(x)\).
  3. Prove the recurrence relation for Hermite polynomials: \(H'_n(x) = 2nH_{n-1}(x)\).
  4. Derive the formula for the Fourier coefficient \(a_n\) for a function \(f(x)\) defined in the interval \([-\pi, \pi]\).
  5. Show that \(\text{erf}(x) + \text{erfc}(x) = 1\), starting from their definitions.
  6. Using the generating function for Legendre polynomials, prove the recurrence relation: \((n+1)P_{n+1}(x) = (2n+1)xP_n(x) - nP_{n-1}(x)\).
  7. Define the Beta function and prove its symmetry property: \(\beta(m,n) = \beta(n,m)\).
  8. Using the Frobenius method, find the indicial equation for Bessel's differential equation: \(x^2y'' + xy' + (x^2 - \nu^2)y = 0\).
Part C: Major Proofs & Derivations
(Answer any FOUR questions | 4 × 8 = 32 Marks)
  1. State and prove the orthogonality property for the Legendre Polynomials: \[ \int_{-1}^{1} P_m(x) P_n(x) \, dx = \frac{2}{2n+1}\delta_{mn} \]
  2. Derive the relation between the Beta and Gamma functions: \[ \beta(m,n) = \frac{\Gamma(m)\Gamma(n)}{\Gamma(m+n)} \]
  3. Discuss the method of separation of variables and use it to derive the general solution for the one-dimensional heat equation \(\frac{\partial u}{\partial t} = c^2 \frac{\partial^2 u}{\partial x^2}\) for a rod of length \(L\) with its ends kept at zero temperature.
  4. State and prove the two most common recurrence relations for Legendre Polynomials involving derivatives:
    1. \( (2n+1)P_n(x) = P'_{n+1}(x) - P'_{n-1}(x) \)
    2. \( nP_n(x) = xP'_n(x) - P'_{n-1}(x) \)

Comprehensive Set 1

Subject: Physics (Mathematical Physics) | Total Marks: 100
Part A
(Answer any NINE questions | 9 × 2 = 18 Marks)
  1. Evaluate the Beta function \(\beta(3, 2.5)\).
  2. What is the value of the Associated Legendre Polynomial \(P_3^1(x)\)?
  3. Write the power series expansion for the error function, \(\text{erf}(x)\).
  4. Evaluate the integral \(\int_0^\infty x^4 e^{-x} \, dx\).
  5. What are Dirichlet's conditions for the convergence of a Fourier series?
  6. State the relationship between the error function (\(\text{erf}\)) and complementary error function (\(\text{erfc}\)).
  7. Find the value of the product \(\Gamma(\frac{1}{4}) \Gamma(\frac{3}{4})\).
  8. Write Laplace's equation in 3D Cylindrical Coordinates.
  9. What is the value of \(P_n'(1)\) for a Legendre polynomial?
Part B
(Answer any EIGHT questions | 8 × 5 = 40 Marks)
  1. Convert the combination \(5H_3(x) + 2H_2(x) - 4H_1(x) + 2H_0(x)\) into a standard polynomial in powers of \(x\).
  2. Using the orthogonality of Legendre polynomials, evaluate: \(\int_{-1}^{1} (P_0(x) + 2P_2(x) + P_3(x)) P_2(x) \,dx\).
  3. Show the parity relation for Associated Legendre Polynomials: \(P_n^m(-x) = (-1)^{n+m} P_n^m(x)\).
  4. Find the Fourier series expansion for the function \(f(x) = x\) in the interval \(-\pi < x < \pi\).
  5. Prove the Hermite polynomial recurrence relation: \(H_n'(x) = 2nH_{n-1}(x)\).
  6. Find and classify the singular points of the differential equation \(x(x-1)^3 y'' + 2(x-1)^3 y' + 4y = 0\).
  7. Prove the recurrence relation for the Beta function: \(\beta(p, q+1) = \frac{q}{p+q} \beta(p, q)\).
  8. Using Rodrigues' formula, derive the expression for the Legendre polynomial \(P_3(x)\).
Part C
(Answer any FOUR questions | 4 × 8 = 32 Marks)
  1. Obtain the Fourier series for \(f(x) = x+x^2\) over \(-\pi < x < \pi\). Hence, show that \(\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}\).
  2. Derive the expression for the electrostatic potential \(V\) of a conducting sphere of radius \(a\) in a uniform external electric field \(\vec{E_0}\). Hence, find the induced surface charge density.
  3. Solve the PDE \(2\frac{\partial u}{\partial t} = 4\frac{\partial u}{\partial x}\) by separation of variables, with initial condition \(u(x,0) = 9e^{-2x}\).
  4. Define Gamma and Beta functions and derive \(\beta(m,n) = \frac{\Gamma(m)\Gamma(n)}{\Gamma(m+n)}\). Using this, prove: \[\int_0^{\pi/2} \sin^p\theta \cos^q\theta \, d\theta = \frac{\Gamma(\frac{p+1}{2})\Gamma(\frac{q+1}{2})}{2\Gamma(\frac{p+q+2}{2})}\]

Comprehensive Set 2

Subject: Physics (Mathematical Physics) | Total Marks: 100
Part A
(Answer any NINE questions | 9 × 2 = 18 Marks)
  1. What is the value of \(\Gamma(-\frac{1}{2})\)?
  2. Write down the generating function for Hermite polynomials, \(H_n(x)\).
  3. If a function \(f(x)\) is even, what can you say about its Fourier sine coefficients (\(b_n\))?
  4. Define a regular singular point for a second-order linear differential equation.
  5. Evaluate the integral \(\int_{-1}^{1} x^3 P_3(x) \,dx\) using properties of Legendre polynomials.
  6. State Parseval's identity for Fourier series.
  7. What is the value of the Legendre polynomial \(P_n(-1)\)?
  8. State the one-dimensional heat conduction equation.
  9. Show that \(\Gamma(n) = \int_0^1 \left(\log \frac{1}{y}\right)^{n-1} \, dy\).
Part B
(Answer any EIGHT questions | 8 × 5 = 40 Marks)
  1. Express the polynomial \(f(x) = 4x^3 - 2x^2 - 3x + 8\) in terms of Legendre polynomials.
  2. Using the Frobenius method, obtain the indicial equation for the differential equation \(x^2y'' - y = 0\).
  3. Prove the Legendre polynomial recurrence relation: \((n+1)P_{n+1}(x) = (2n+1)xP_n(x) - nP_{n-1}(x)\).
  4. Find the half-range sine and cosine series for the function \[f(x) = \begin{cases} 0, & -\pi < x < 0 \\ x, & \;\; 0 \le x < \pi \end{cases}\]
  5. Show that for Bessel functions, \(J_{1/2}(x) = \sqrt{\frac{2}{\pi x}}\sin x\).
  6. Using the generating function for Legendre polynomials, prove: \((2n+1)P_n(x) = P'_{n+1}(x) - P'_{n-1}(x)\).
  7. Evaluate \(\int_{-1}^1 x P_n(x) P_{n-1}(x) \,dx\) using a recurrence relation.
  8. Derive the solution to Laplace's equation \(\nabla^2 u = 0\) in a 2D rectangular region \(0 \le x \le a, 0 \le y \le b\).
Part C
(Answer any FOUR questions | 4 × 8 = 32 Marks)
  1. Solve the 1D wave equation \(\frac{\partial^2 y}{\partial t^2} = c^2 \frac{\partial^2 y}{\partial x^2}\) with boundary conditions \(y(0,t)=0, \ y(L,t)=0\) and initial conditions \(y(x,0) = f(x)\) and \(\frac{\partial y}{\partial t}\Big|_{(x,0)} = g(x)\).
  2. State and prove the orthogonality property for Hermite polynomials: \[\int_{-\infty}^{\infty} e^{-x^2} H_m(x) H_n(x) \,dx = 2^n n! \sqrt{\pi} \, \delta_{mn}\]
  3. Derive the formulas for the Fourier coefficients \(a_0, a_n, b_n\) for a function \(f(x)\) in the interval \([-L, L]\).
  4. Using the Frobenius method, find the complete polynomial solution of the Legendre equation \((1-x^2)y'' - 2xy' + \lambda y = 0\), where \(\lambda = n(n+1)\) and n is an integer.
