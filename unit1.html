Unit 1: Computer Fundamentals

1. Introduction to Computer

A computer is an electronic device that manipulates information, or data. It has the ability to store, retrieve, and process data according to a set of instructions. Computers are used for a vast range of tasks, from simple document creation to complex scientific calculations.

1.1 Generations of Computers

The history of computers is categorized into generations, each marked by a significant technological advancement.

Generation Period Key Technology Characteristics
First 1940-1956 Vacuum Tubes Very large, high power consumption, unreliable, used machine language. E.g., ENIAC.
Second 1956-1963 Transistors Smaller, faster, cheaper, more reliable than first-gen. Used assembly language. E.g., IBM 1401.
Third 1964-1971 Integrated Circuits (ICs) Increased speed and efficiency, keyboards and monitors introduced. E.g., IBM-360 series.
Fourth 1971-Present Microprocessors (VLSI) Thousands of ICs on a single chip. Led to personal computers (PCs). E.g., Apple Macintosh.
Fifth Present & Beyond Artificial Intelligence (AI) & ULSI Focus on parallel processing, AI, expert systems, and natural language.

1.2 Types of Computers

  • Supercomputer: The most powerful computers, used for complex tasks like weather forecasting and scientific research.
  • Mainframe Computer: Large, powerful computers used by corporations for bulk data processing and transaction handling.
  • Minicomputer: A multi-user computer, smaller than a mainframe but larger than a microcomputer. Often used as a server.
  • Microcomputer (PC): Designed for individual use. Includes desktops, laptops, tablets, and smartphones.

2. Computer System Hardware

Hardware refers to the physical components of a computer system.

2.1 Block Diagram of a Computer

A computer system consists of an Input Unit, a Central Processing Unit (CPU), a Memory Unit, and an Output Unit. The CPU is the core, containing the Control Unit (CU) and the Arithmetic Logic Unit (ALU).

  • Input Unit: Accepts data from the user (e.g., keyboard, mouse).
  • CPU: Processes data. The CU directs operations while the ALU performs calculations.
  • Memory Unit: Stores data and instructions.
  • Output Unit: Presents results to the user (e.g., monitor, printer).

3. Computer Memory

Memory is where the computer holds data and instructions for processing.

3.1 Memory Hierarchy

Memory is organized in a hierarchy based on speed, cost, and capacity. Faster memory is more expensive and has less capacity.

  • CPU Registers: Fastest, smallest memory, located inside the CPU itself.
  • Cache Memory: A very high-speed buffer between the CPU and main memory (RAM). Stores frequently used data.
  • Primary Memory (RAM): The computer's main working memory. It is volatile (data is lost on power off).
  • Secondary Memory (Storage): Non-volatile storage for long-term data retention (e.g., HDD, SSD, USB Drive).

4. Input & Output Devices

4.1 Input Devices

  • Keyboard & Mouse: Standard devices for user interaction.
  • Scanner: Converts physical documents into digital format.
  • Microphone: Captures audio input.

4.2 Output Devices

  • Monitor (VDU): Displays visual information.
  • Printer: Creates hard copies of documents.
  • Speakers: Produce audio output.

5. Computer Software

Software is a set of instructions that tells the hardware what to do.

  • System Software: Manages computer hardware and provides a platform for applications (e.g., Windows, Linux, Device Drivers).
  • Application Software: Programs designed for end-users to perform specific tasks (e.g., MS Word, Chrome, Photoshop).
  • Utility Software: Helps to maintain and configure the computer (e.g., Antivirus, Disk Cleanup).
  • Firmware: Permanent software programmed into a device's read-only memory (e.g., BIOS).

6. Computer Network

A computer network is a collection of interconnected computers for sharing resources and information.

6.1 Network Types

  • LAN (Local Area Network): Covers a small geographical area like an office or home.
  • MAN (Metropolitan Area Network): Spans a city, connecting multiple LANs.
  • WAN (Wide Area Network): Covers a large area, like a country. The Internet is the largest WAN.

6.2 Network Devices

  • Repeater: Amplifies a signal to extend its range.
  • Hub: A basic connector that broadcasts data to all connected devices.
  • Bridge: Connects two LANs and filters traffic based on MAC addresses.
  • Switch: A smart hub that sends data only to the intended recipient device.
  • Router: Connects different networks and forwards data packets between them using IP addresses. It's the key to the internet.
  • Gateway: A node that connects two networks with different protocols, acting as an entry/exit point.
