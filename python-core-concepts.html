2. What are the advantages of Python programming language?

The main advantages of Python are:

  • Easy to Read and Learn: Its syntax is simple and clean, resembling plain English.
  • Large Standard Library: It includes a vast collection of modules and functions for common tasks.
  • Cross-Platform: Python code can run on different operating systems (Windows, macOS, Linux) without modification.
  • Interpreted Language: Code is executed line by line, which simplifies debugging and allows for rapid development.

3. What are tokens in Python? How many types of tokens are allowed in Python? Exemplify your answer.

A token is the smallest individual unit in a Python program that is meaningful to the interpreter. Python has five types of tokens:

  1. Keywords: Reserved words like `if`, `for`, `def`.
  2. Identifiers: Names for variables, functions, etc., like `my_variable`.
  3. Literals: Constant data values like `101` or `"hello"`.
  4. Operators: Symbols that perform operations, like `+`, `-`, `*`, `=`.
  5. Punctuators: Symbols for structure, like `()`, `{ }`, `,`, `#`.

Example: In the statement `x = 10`, the tokens are `x` (Identifier), `=` (Operator), and `10` (Literal).

8. How are string-literals represented and implemented in Python?

Representation: String-literals in Python can be enclosed in single quotes (`'hello'`), double quotes (`"world"`), or triple quotes for multi-line strings (`'''This is a multi-line string'''`).

Implementation: Internally, Python strings are implemented as **immutable sequences** of Unicode characters. This immutability means that once a string is created, it cannot be changed. Any operation that appears to modify a string actually creates a new string object in memory.

12. What is a variable? How are they important for a program?

A variable is a named location in the computer's memory used to store data. You can assign a value to a variable and refer to it later by its name.

Importance: Variables are crucial because they allow a program to store and manipulate data. They make code readable and manageable by giving descriptive names to values (e.g., `user_age = 25` is more understandable than just `25`). They also enable programs to work with data that can change during execution.

15. What is the significant of pass statement?

The `pass` statement in Python is a null operation; nothing happens when it executes. Its significance is that it acts as a **placeholder**. It is used when a statement is syntactically required, but you have no code to write, preventing syntax errors in empty function definitions, classes, or `if` blocks.

python
def my_future_function():
    pass  # To avoid a syntax error for an empty function body

17. What is type casting? How it is used?

Type casting (or type conversion) is the process of explicitly converting a value from one data type to another. It is used by calling the target data type's constructor function (like `int()`, `float()`, or `str()`) on a value.

python
num_string = "100"       # A string
num_integer = int(num_string) # Casting string to integer
print(num_integer + 5)   # Output: 105

5. Write the use of various string function to work with string values.

Python provides a rich set of built-in functions and methods to manipulate strings. Some of the most common ones are:

  1. `len()` function: Returns the number of characters in a string.
  2. `.upper()` and `.lower()` methods: Convert a string to all uppercase or all lowercase letters.
  3. `.strip()` method: Removes any leading (start) and trailing (end) whitespace characters.
  4. `.split(separator)` method: Splits a string into a list of substrings based on a specified separator.
  5. `.replace(old, new)` method: Returns a new string where a specified value is replaced with another.

7. What is for loop. Write with an example to print first 20 natural numbers.

A `for` loop in Python is an iteration construct used to execute a block of code repeatedly for each item in a sequence (such as a list, tuple, or string). It is ideal for situations where you know in advance how many times you want to loop.

Example: Program to print the first 20 natural numbers
python
print("The first 20 natural numbers are:")
for number in range(1, 21):
    print(number)

9. What is conditional statement. Write with an example with multiple conditional statements.

A conditional statement is a programming construct that allows a program to execute different blocks of code based on whether a specific condition evaluates to `True` or `False`. In Python, the primary conditional statements are `if`, `elif` (else if), and `else`.

Example with multiple conditional statements (Grading System):
python
score = int(input("Enter the student's score: "))

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print(f"The student's grade is: {grade}")
