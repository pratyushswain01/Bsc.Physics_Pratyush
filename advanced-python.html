4. What is tuple and how to unpack it?

A tuple is an ordered, immutable collection of elements in Python. Unpacking a tuple means assigning its items to multiple variables in a single statement. The number of variables must match the number of items.

python
# Create a tuple
point = (10, 20, 30)

# Unpack the tuple
x, y, z = point
print(f"x: {x}, y: {y}, z: {z}")

19. What is an object? How to use it?

An object is an instance of a class. It represents a real-world entity and bundles data (attributes) and behavior (methods) together. To use an object, you first create it (instantiation) and then access its attributes and call its methods using the dot (`.`) operator.

python
my_list = [1, 2, 3]  # my_list is an object of the 'list' class
my_list.append(4)    # Using the 'append' method of the list object

20. What is constructor? How to create it in python?

A constructor is a special method in a class that is automatically called when a new object is created. Its primary purpose is to initialize the object's attributes. In Python, the constructor is always named `__init__()`.

python
class Car:
    # Constructor method
    def __init__(self, color):
        self.color = color  # Initializing an attribute

# Creating an object calls the constructor
my_car = Car("Red")
print(my_car.color) # Output: Red

1. What is list and write the use of following functions. Append(), extend(), sort(), reverse(), index()

A list is an ordered and mutable data collection in Python that can store a sequence of elements. Its elements can be changed, added, or removed after creation.

  • `append(element)`: Adds a single element to the end of the list.
  • `extend(iterable)`: Adds all elements of an iterable (like another list) to the end.
  • `sort()`: Sorts the list's elements in place (by default, ascending).
  • `reverse()`: Reverses the order of elements in the list in place.
  • `index(element)`: Returns the index of the first occurrence of the specified element.

2. How to create a user define function? Explain with an example to find the highest of two integer value using function.

A user-defined function is a block of reusable code created using the `def` keyword, followed by the function name, parentheses `()` for parameters, and a colon `:`. The code block inside must be indented.

python
# 1. Define the function
def find_highest(a, b):
    """This function takes two numbers and returns the highest."""
    if a > b:
        return a
    else:
        return b

# 2. Call the function
highest_value = find_highest(10, 25)
print(f"The highest number is: {highest_value}")

3. What is nested loop? Write with an example using while loop.

A nested loop is a loop placed inside another loop. For each single iteration of the outer loop, the inner loop completes its full set of iterations. This structure is very useful for working with two-dimensional data structures like matrices or for creating patterns.

Example: Printing a Multiplication Table using Nested `while` Loops
python
# Outer loop controls the rows (from 1 to 5)
i = 1
print("Multiplication Table (5x5):")
print("---------------------------")
while i <= 5:
    # Inner loop controls the columns (from 1 to 5)
    j = 1
    while j <= 5:
        product = i * j
        print(f"{product:3}", end=" ") # Format for alignment
        j += 1
    print() # Move to the next line
    i += 1

4. Create a dictionary book to store some book details like book code, name, price and author. Input any book code and print the book name, price and author details.

To store details for multiple books, we use a nested dictionary where the main dictionary's keys are book codes. The value for each key is another dictionary containing that book's details.

python
# 1. Create a nested dictionary to store book details
books = {
    "B101": {"name": "The Python Handbook", "price": 499.00, "author": "A. Sharma"},
    "B102": {"name": "Data Structures in C", "price": 650.50, "author": "R. Verma"}
}

# 2. Prompt the user to enter a book code
input_code = input("Enter Book Code: ")

# 3. Check if the code exists and print details
if input_code in books:
    details = books[input_code]
    print("\n--- Book Details Found ---")
    print(f"Name: {details['name']}")
    print(f"Author: {details['author']}")
    print(f"Price: ₹{details['price']:.2f}")
else:
    print(f"\nSorry, book with code '{input_code}' not found.")

5. Create a class employee having data members empno, ename and salary. Write a constructor to assign default value. Write a function to calculate net salary. Net salary = salary + 20% bonus on salary. Write another function to print the employee details including net salary.

python
class Employee:
    """A class to represent an employee."""

    # 1. Constructor with default values
    def __init__(self, empno=0, ename="N/A", salary=0.0):
        self.empno = empno
        self.ename = ename
        self.salary = salary

    # 2. Method to calculate net salary
    def calculate_net_salary(self):
        bonus = self.salary * 0.20
        return self.salary + bonus

    # 3. Method to print all details
    def display_details(self):
        net_sal = self.calculate_net_salary()
        print("--- Employee Details ---")
        print(f"Employee Number: {self.empno}")
        print(f"Employee Name:   {self.ename}")
        print(f"Base Salary:     ₹{self.salary:.2f}")
        print(f"Net Salary (w/Bonus): ₹{net_sal:.2f}")
        print("------------------------\n")

# --- Demonstrate the class ---
emp1 = Employee(101, "Ravi Kumar", 50000)
emp1.display_details()

emp_default = Employee()
emp_default.display_details()
