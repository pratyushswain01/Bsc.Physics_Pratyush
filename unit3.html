Unit 3: Introduction to Python

1. Getting Started with Python

Python is a high-level, interpreted programming language known for its simple, readable syntax. It was created by Guido van Rossum and released in 1991. Its design philosophy emphasizes code readability, making it an excellent choice for beginners and for rapid application development.

The Zen of Python: You can see the guiding principles of Python by typing import this into a Python interpreter. The first line is "Beautiful is better than ugly."

2. Python Basics

Identifiers and Variables

An identifier is a name given to an entity like a variable, function, or class. Rules for identifiers:

  • Can be a combination of letters (a-z, A-Z), digits (0-9), and underscore (_).
  • Cannot start with a digit.
  • Are case-sensitive (age and Age are different).
  • Cannot be a keyword.

A variable is a symbolic name that is a reference or pointer to an object. Once an object is assigned to a variable, you can refer to the object by that name.

# Variable assignment
age = 25              # age is an integer variable
name = "Alice"        # name is a string variable
is_student = True     # is_student is a boolean variable
pi_value = 3.14       # pi_value is a float variable

print(name, "is", age, "years old.")

Keywords

Keywords are reserved words in Python that have special meaning. They cannot be used as variable names. There are 35 keywords in Python 3.8.

Examples: if, else, for, while, def, class, True, False, None, import.

Comments, Indentation, and Multi-Lining

  • Comments: Start with #. They are ignored by the interpreter and are used for explaining code.
  • Indentation: Python uses indentation (whitespace at the beginning of a line) to define code blocks, unlike other languages that use braces {}. This is a strict rule.
  • Multi-lining: A long statement can be split into multiple lines using the backslash \ character.
# This is a single-line comment

# Correct Indentation defines a code block
if age > 18:
    print("You are an adult.") # This line is inside the if-block

# Multi-line statement
total_sum = 1 + 2 + 3 + \
            4 + 5 + 6
print(total_sum) # Output: 21

3. Data Types and Variables

Basic (Primitive) Types

  • Integer (int): Whole numbers, like 10, -50, 0.
  • Float (float): Numbers with a decimal point, like 9.81, -0.001.
  • String (str): Sequence of characters, enclosed in single ', double ", or triple ''' quotes.
  • Boolean (bool): Represents truth values, can only be True or False.

Mutable vs. Immutable Types

This is a fundamental concept in Python.

  • Immutable Types: Their value/state cannot be changed after creation. If you change them, a new object is created in memory. Examples: int, float, bool, str, tuple.
  • Mutable Types: Their value/state can be changed in-place after creation. Examples: list, dict, set.
# --- Immutable Example (str) ---
my_string = "hello"
print(id(my_string))  # Print memory address, e.g., 2206716501360

my_string = my_string + " world" # A NEW string object is created
print(my_string) # "hello world"
print(id(my_string)) # A NEW memory address, e.g., 2206716760176

# --- Mutable Example (list) ---
my_list = [1, 2, 3]
print(id(my_list)) # Print memory address, e.g., 2206716616640

my_list.append(4) # The list is modified IN-PLACE
print(my_list) # [1, 2, 3, 4]
print(id(my_list)) # SAME memory address, e.g., 2206716616640

Type Conversion (Casting)

You can convert values from one type to another using built-in functions like int(), float(), str().

num_string = "100"
num_integer = int(num_string)  # Convert string to integer
print(num_integer + 5) # Output: 105

num_float = float(num_integer) # Convert integer to float
print(num_float) # Output: 100.0

4. Operators

Arithmetic Operators

  • + (Addition), - (Subtraction), * (Multiplication), / (Float Division)
  • // (Integer/Floor Division), % (Modulus/Remainder), ** (Exponent)

Operator Precedence & Associativity

Precedence determines which operators are evaluated first. Associativity determines the order for operators with the same precedence.

PrecedenceOperatorDescription
Highest()Parentheses
**Exponent (Right-to-Left)
*, /, //, %Multiplication/Division (Left-to-Right)
Lowest+, -Addition/Subtraction (Left-to-Right)
 
# Precedence example
result = 10 + 2 * 3   # 2 * 3 is done first (precedence)
print(result) # Output: 16

# Associativity example
result = 2 ** 3 ** 2  # Evaluated as 2 ** (3 ** 2) -> 2 ** 9
print(result) # Output: 512

5. Working with Strings

Accessing String Elements (Slicing)

You can access parts of a string using a slice [start:stop:step]. Indices start at 0.

text = "Python is Fun!"
print(text[0])       # 'P' (First character)
print(text[-1])      # '!' (Last character)
print(text[0:6])     # 'Python' (from index 0 up to, but not including, 6)
print(text[7:])      # 'is Fun!' (from index 7 to the end)
print(text[::-1])    # '!nuF si nohtyP' (Reverse the string)

Common String Methods and Functions

  • len(str): A built-in function to get the length of a string.
  • str.upper() / str.lower(): Convert to uppercase/lowercase.
  • str.strip(): Remove leading/trailing whitespace.
  • str.find('sub'): Returns the first index of 'sub'; -1 if not found.
  • str.replace('old', 'new'): Replaces all occurrences of 'old' with 'new'.

6. Console Input & Output

Getting input from the user and displaying output is essential.

Receiving Input and Console I/O

The input() function reads a line from the console, converts it to a string, and returns it.

# I/O Operation
user_name = input("Enter your name: ")
user_age_str = input("Enter your age: ")

# input() always returns a string, so we need to convert it
user_age_int = int(user_age_str)

print("Hello, " + user_name + "!")

Formatted Printing with f-Strings

F-strings (formatted string literals) provide a concise and convenient way to embed expressions inside string literals for formatting.

# Formatting Printing
print(f"Hello, {user_name}! You will be {user_age_int + 1} next year.")
# Example output: Hello, Alice! You will be 26 next year.

pi = 3.14159265
print(f"The value of Pi is approximately {pi:.2f}") 
# .2f formats the float to 2 decimal places
# Output: The value of Pi is approximately 3.14

7. Control Flow Instructions

Decision Control (if, elif, else)

This allows the program to execute different blocks of code based on conditions.

  • Logical Operators: and, or, not
  • Conditional Expression (Ternary Operator): A one-line if-else statement. value_if_true if condition else value_if_false
  • pass Statement: A null operation. It's used as a placeholder where code is syntactically required but you have nothing to write yet.
score = 85

if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
else:
    grade = 'F'
    
print(f"Your grade is {grade}") # Output: Your grade is B

# Ternary operator
status = "Adult" if user_age_int >= 18 else "Minor"
print(f"Status: {status}")

# Pass statement
if user_age_int < 0:
    pass # TODO: Add error handling here later

Repetition Control (Loops)

Loops are used to execute a block of code repeatedly.

  • for loop: Iterates over a sequence (like a list, string, or range).
  • while loop: Repeats as long as a condition is true.
  • break: Immediately exits the current loop.
  • continue: Skips the rest of the current iteration and proceeds to the next one.
  • else block in a loop: Executes only if the loop completes normally (i.e., not terminated by a break statement).
# for loop with range()
for i in range(5):  # 0, 1, 2, 3, 4
    print(i, end=" ") # Output: 0 1 2 3 4
print()

# while loop
count = 5
while count > 0:
    print(count, end=" ") # Output: 5 4 3 2 1 
    count -= 1
print()

# break, continue, and else in a loop
for num in range(2, 10):
    if num == 7:
        continue # Skip printing 7
    if num == 9:
        print("\nFound 9, breaking loop.")
        break    # Exit the loop
    print(num, end=" ")
else:
    print("Loop finished without breaking.") # This line won't run

# Output: 2 3 4 5 6 8 
#         Found 9, breaking loop.

8. Modules and Built-in Functions

Python's power comes from its extensive standard library, accessible through modules.

  • Built-in Functions: Always available, like print(), len(), int(), input(), range().
  • Modules: Files containing Python code that you can import into your script to use their functions and classes. Example: the math module.
# Using a module
import math

radius = 5
area = math.pi * math.pow(radius, 2)
print(f"The area of the circle is {area:.2f}") # Output: The area of the circle is 78.54

# Using built-in all() and any() functions
# all(): returns True if all items in an iterable are true
# any(): returns True if any item in an iterable is true
list1 = [True, True, True]
list2 = [True, False, True]

print(f"all(list1): {all(list1)}") # True
print(f"any(list2): {any(list2)}") # True
print(f"all(list2): {all(list2)}") # False
Objects and Classes (A Glimpse): Your syllabus mentions classes/objects. In Python, everything is an object. An object has properties (data) and methods (functions). For example, a string object has data (the characters) and methods like .upper(). A class is the blueprint for creating objects. You've already been using them!
