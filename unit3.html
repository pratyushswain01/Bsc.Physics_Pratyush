Unit 3: Introduction to Python

1. Introduction to Python

Python is a high-level, interpreted programming language known for its remarkable readability and simple syntax. This makes it an ideal first language for beginners and a powerful tool for rapid development for experts. In Python, you write code that looks close to plain English, focusing on solving problems rather than wrestling with complex syntax.

2. Python Basics: The Building Blocks

Identifiers, Keywords, and Variables

  • Identifiers: Names for variables, functions, etc. Must start with a letter or underscore, followed by letters, numbers, or underscores. They are case-sensitive (Age is not age).
  • Keywords: Reserved words with special meanings like if, for, def, class. You cannot use them as identifiers.
  • Variables: Labels for data. Python is dynamically typed, so you don't declare types. The type is inferred when you assign a value.

Comments, Indentation, & Multi-lining

  • Comments: Use # for single-line comments. They are crucial for explaining your code's logic.
  • Indentation: Python's signature feature. It's not optional; it defines code blocks (like inside an if statement or a loop). Four spaces per indentation level is the standard.
  • Multi-lining: For long lines of code, use a backslash \ to continue a statement on the next line.
python
# This is a comment. It explains the code.
user_name = 'Pratyush'  # A variable with an identifier 'user_name'

if len(user_name) > 0:
    print(f"Welcome, {user_name}!") # This line is indented, so it's inside the 'if' block.

# A multi-line statement for readability
total_calculation = (100 + 200 + 300 + 
                     400 + 500)

Objects in Python

Everything is an Object: A core concept. Every value in Python—a number, a string, a list, even a function—is an object. An object bundles data (attributes) and functionality (methods) together. When you write user_name.upper(), you are calling the upper() method on the string object stored in `user_name`. A class is the blueprint used to create these objects.

3. Python Types & Operators

Basic & Container Types

  • Basic Types: The simplest data forms. int (e.g., 10), float (e.g., 3.14), str (e.g., "hello"), bool (True or False).
  • Container Types: Hold collections of other objects. Main examples are list, tuple, dict, and set (covered in Unit 4).

Mutable vs. Immutable Types

This critical distinction affects how you work with data.

  • Immutable: Value cannot be changed after creation. All Basic Types are immutable. Any "change" creates a brand new object in memory.
  • Mutable: Value can be changed in-place without creating a new object. Lists and Dictionaries are mutable.

Arithmetic Operators, Precedence, & Associativity

These define the rules of mathematical and logical operations.

  • Operators: +, -, *, / (float division), // (floor division), % (modulus), ** (exponent).
  • Precedence: The order of operations. ** is first, then * / % //, then + -. Parentheses () override all.
  • Associativity: Order for operators with the same precedence. Most are left-to-right (e.g., 10-5-2 is (10-5)-2). The exponent operator ** is right-to-left (2**3**2 is 2**(3**2)).
python
# Precedence example: multiplication before addition
result = 10 + 5 * 2  # 10 + 10
print(result)  # Output: 20

# Using parentheses to control the order
result_controlled = (10 + 5) * 2  # 15 * 2
print(result_controlled) # Output: 30

# Floor division (discards the remainder)
print(10 // 3) # Output: 3

Type Conversion (Casting)

You can explicitly change a value's type using functions like int(), float(), and str().

python
age_str = "25" # Data from an input form is often a string
age_int = int(age_str) # Convert to an integer for calculations

print(f"In five years, you will be {age_int + 5} years old.")

4. Deep Dive into Strings

Strings are one of the most used types. They are immutable sequences of characters.

Accessing, Slicing, and Properties

  • Accessing: Use square brackets [] with an index (starting from 0). text[0] is the first character.
  • Slicing: Extract a portion of the string with [start:stop:step].
  • Built-in Functions/Methods: len() gives the length. Methods like .upper(), .lower(), .strip(), .find(), and .replace() provide powerful functionality. Remember, these methods return a *new* string; they don't change the original.
  • Comparison: Strings are compared lexicographically (like in a dictionary). 'a' < 'b' is True.
python
quote = "   Learning Python is rewarding!   "

# Clean up the string using a method
cleaned_quote = quote.strip()  # Removes leading/trailing whitespace
print(cleaned_quote)

# Replace a word
modified_quote = cleaned_quote.replace('rewarding', 'empowering')
print(modified_quote)

# Find the start index of a word
position = modified_quote.find('Python')
print(f"'Python' starts at index: {position}")

5. I/O and Control Flow

Console I/O and Formatting

  • Input: The input("prompt") function gets user input from the console. It always returns a string.
  • Output: The print() function displays information.
  • Formatting: **F-strings** (f"...") are the modern, standard way to embed expressions directly inside strings. They are fast, readable, and powerful.

Decision Control (if/elif/else)

This is the core of your program's logic. It allows you to execute specific code blocks based on whether a condition is true or false.

Conditional Expression (Ternary Operator)

A compact, one-line version of an `if/else` statement. It's great for simple assignments.

python
age = int(input("Please enter your age: "))

# Ternary operator for a concise assignment
status = "Adult" if age >= 18 else "Minor"
print(f"Your status is: {status}")

# Full if/elif/else structure for complex logic
if status == "Adult":
    print("You can vote and drive.")
elif age > 12:
    print("You are a teenager.")
else:
    print("You are a child.")

6. Repetition: The Power of Loops

Loops are fundamental for automating tasks and processing collections of data without repeating code.

Looping Instructions: for & while

  • for loop: The "definite" loop. Use it when you want to iterate over a sequence (like a list, a string, or a range of numbers) a specific number of times.
  • while loop: The "indefinite" loop. Use it when you want to repeat a block of code as long as a certain condition remains true.

Controlling Loop Behavior: break, continue, pass, and else

  • break: Exits the loop immediately, regardless of the loop's condition.
  • continue: Skips the rest of the current iteration and jumps to the beginning of the next one.
  • pass: A null statement. It's a placeholder used when a statement is syntactically required but you don't want any code to execute.
  • else block in a loop: This is a unique Python feature. The `else` block executes *only if the loop completes its entire sequence naturally* (i.e., it was not terminated by a `break` statement). This is incredibly useful for search operations.
python
search_target = 7
print(f"Searching for the number {search_target}...")

for number in range(1, 11):
    if number == search_target:
        print(f"Found it! Number {number}.")
        break # Exit the loop since we found what we wanted
else:
    # This block only runs if the 'for' loop finishes without a 'break'.
    print(f"The number {search_target} was not found in the range.")

7. Modules and Built-in Functions

Python's "batteries-included" philosophy means it comes with a vast Standard Library, organized into **modules**.

  • Modules: Python files containing functions, classes, and variables that you can import and use in your code. The math module is a classic example.
  • Built-in Functions: A core set of functions that are always available without needing to import anything. You've been using them all along: print(), len(), int(), range(), etc.
  • all() and any(): These are powerful built-in functions for working with iterables (like lists). all() returns True if every element is true. any() returns True if at least one element is true.
python
# 1. Using a module from the standard library
import math

print(f"The value of PI is: {math.pi}")
print(f"The square root of 16 is: {math.sqrt(16)}")

# 2. Using the all() and any() built-in functions
permissions = [True, True, False, True]

# Does the user have full permissions?
has_all_permissions = all(permissions) # False, because one is False
print(f"User has all permissions: {has_all_permissions}")

# Does the user have at least one permission?
has_some_permissions = any(permissions) # True
print(f"User has some permissions: {has_some_permissions}")
